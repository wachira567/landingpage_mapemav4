# Mapema 151515 Ltd — Enterprise Landing Page

A modern, B2B-focused landing page for Mapema 151515 Ltd, a Nairobi-based internet infrastructure and connectivity provider serving enterprises across Kenya.

## About Mapema 151515 Ltd

Mapema 151515 Ltd is a Kenyan technology company headquartered in Westlands, Nairobi. Originally established as a consumer rewards platform, the company has evolved to deliver enterprise-grade internet infrastructure including dedicated fiber links, MPLS private networks, cloud connectivity, managed WiFi, data center colocation, and 24/7 network security operations.

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- GSAP (animations + ScrollTrigger)
- Lenis (smooth scrolling)
- WebGL (hero waveform visualization)
- shadcn/ui components

## Key Features

- **Interactive WebGL Hero**: Animated waveform visualization with mouse-reactive dot field
- **B2B Enterprise Focus**: 80% enterprise messaging targeting CTOs, IT managers, and business owners
- **Lead Generation Forms**: "Request Quote" flow with service selection — no consumer checkout
- **Scroll-Triggered Animations**: Staggered text reveals, counter animations, card entrances
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile
- **Network Coverage Visualization**: Animated SVG map of Nairobi business districts

## Development

```bash
# Build for production

```
